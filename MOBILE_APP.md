# Native mobile app

The Expo (Android / iOS) app is maintained in a **separate Git repository** so it does not share `node_modules` with this Vite web app (avoids duplicate React / Metro issues).

**Location on disk:** `../dreemystar-mobile` (sibling of this `project` folder).

Open that folder in its own Cursor/VS Code window and run `git` / `npm` from there.
